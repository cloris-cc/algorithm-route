使用场景：**数组**由于出现负数等情况**无法使用滑动窗口**的情况下，可考虑使用前缀和。

前缀和只是一个统称。除了和外也可以是乘、除等数学运算；除了前缀也可以是后缀。

前缀和与哈希表的联系：使用map来存放前后缀的好处是**不需要考虑前/后缀变量的长度/空间大小**。

下面是计算左前缀，右后缀的通用代码模板：
```java
        int len = nums.length;
        int[] L = new int[len];
        int[] R = new int[len];

        L[0] = 0;
        R[len - 1] = 0;
        for (int i = 1; i < len; i++) {
            L[i] = L[i - 1] + nums[i - 1];
        }
        for (int i = len - 2; i >= 0; i--) {
            R[i] = R[i + 1] + nums[i + 1];
        }
        // 例如，nums[i]的左前缀和为L[i],右后缀和为R[i]
```
